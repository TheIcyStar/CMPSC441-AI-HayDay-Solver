## Development
1. Ensure NodeJS is installed on your system
```sh
node --version
```

2. Install dependencies
```sh
npm install
```

3. Build the project
```sh
npm run build
```

4. Run the project
```sh
npm run start
```

### Debugging with VSCode
You can do steps 3 and 4 by pressing F5 from your editor.
